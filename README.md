# eMart Frontend

eMart Frontend is a responsive e-commerce web application frontend built using React and Vite.  
It provides a clean user interface for an online shopping platform and is designed to integrate easily with backend REST APIs.

## Features
- Responsive and user-friendly UI
- Component-based architecture using React
- Fast build and development using Vite
- Clean and scalable project structure
- Ready for backend integration

## Tech Stack
- React
- JavaScript
- Vite
- HTML & CSS
- Git & GitHub

emartfrontend/

  - pages/

  - main.jsx

- vite.config.js

## Installation & Run Locally

1. Clone the repository  
   git clone https://github.com/Rohith101coder/emartfrontend.git

2. Go to project directory  
   cd emartfrontend

3. Install dependencies  
   npm install

4. Start development server  
   npm run dev

The app will run on: http://localhost:5173/

## Backend Integration
This frontend is designed to work with a backend e-commerce system that manages:
- Products
- Users
- Cart and orders
- Authentication
